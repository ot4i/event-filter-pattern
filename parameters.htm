Parameters for the Event Filter pattern

IBM MessageSight Endpoint Configuration
Pattern parameter Default Description
Server

The host name or IP address of the IBM MessageSight server
Port

The port number of the IBM MessageSight Endpoint
Event Filtering
Pattern parameter Default Description
Topic /PACEMAKER/#

The topic used by the flow to subscribe for inbound messages
Filter contains($Body/Data/event/type,"alert")

An XPath value used to filter the inbound messages
Filtered Event Output
Pattern parameter Default Description
Queue manager name

The Queue Manager used to store filtered messages
Queue name EVENT.IN

The Queue used to store filtered messages
Error Handling
Pattern parameter Default Description
Error message required Selected

Specifies whether an error message is required if an error occurs in the flow
Error queue manager

The Queue Manager used to store error messages
Error Queue ERROR

The Queue used to store error messages

All the pattern parameters are listed with their default values. Where no default is indicated, the default value is an empty string.
