「IBM MessageSight インバウンド・イベント・フィルター」パターンを適用する前に完了するタスク

パターン・インスタンスの生成のためにパターンを適用する前に、以下のステップを実行します。

  • IBM MessageSight クライアント JAR ファイルを IBM Integration Bus マシンにコピーします。
  • IBM Integration Console ウィンドウで以下のコマンドを実行することにより、IBM MessageSight クライアント JAR ファイルの場所に基づいて IBM MessageSight 構成可能サービスを更新します。 

    mqsichangeproperties node_name -c JMSProviders 
        	-o IBM_MessageSight -n jarsURL -v client_jar_location

    ここで、node_name は統合ノード (ブローカー) の名前 (例えば IB9NODE)、client_jar_location は IBM MessageSight クライアント JAR ファイルのパス (例えば C:\ImaClient\jms\lib) です。

  • 統合ノードを再始動します。
