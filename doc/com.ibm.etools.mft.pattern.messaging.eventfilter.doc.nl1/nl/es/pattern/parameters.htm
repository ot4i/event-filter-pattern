Parámetros para el patrón Filtro de sucesos

Configuración de punto final de IBM MessageSight
Parámetro de patrón Predeterminado Descripción
Servidor

El nombre de host o dirección IP del servidor de IBM MessageSight

Puerto

El número de puerto del punto final de IBM MessageSight

Filtrado de eventos
Parámetro de patrón Predeterminado Descripción
Tema /PACEMAKER/#

El tema utilizado por el flujo para suscribirse a los mensajes de entrada

Filtro contains($Body/Data/event/type,"alert")

Un valor de XPath utilizado para filtrar los mensajes de entrada

Salida de sucesos filtrados
Parámetro de patrón Predeterminado Descripción
Nombre del gestor de colas

El gestor de colas utilizado para almacenar mensajes filtrados

Nombre de cola EVENT.IN

La cola utilizada para almacenar mensajes filtrados

Manejo de errores
Parámetro de patrón Predeterminado Descripción
Mensaje de error necesario Seleccionado

Especifica si un mensaje de error es necesario si se produce un error en el flujo

Gestor de colas de errores

El gestor de colas utilizado para almacenar mensajes de error

Cola de errores ERROR

La cola utilizada para almacenar mensajes de error

Todos los parámetros de patrón aparecen en la lista con sus valores predeterminados. Cuando no se indique un valor predeterminado, éste es una serie vacía.
