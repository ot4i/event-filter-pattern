Restricciones en el uso del patrón Filtro de sucesos de entrada de IBM MessageSight

Se aplican una serie de restricciones cuando se utiliza este patrón.

Puede utilizar el patrón Filtro de sucesos de entrada de IBM MessageSight sólo si se cumplen todas las restricciones siguientes:

  • Hay un dispositivo o una imagen virtual de IBM MessageSight disponible.
