Tareas que deben completarse antes de aplicar el patrón Filtro de sucesos de entrada de IBM MessageSight

Antes de aplicar el patrón para generar una instancia de patrón, realice los pasos siguientes:

  • Copie los archivos JAR de cliente de IBM MessageSight a la máquina de IBM Integration Bus.
  • Actualice el servicio configurable de IBM MessageSight con la ubicación de los archivos JAR de cliente de IBM MessageSight ejecutando el mandato siguiente en una ventana de IBM Integration Console: 

    mqsichangeproperties nombre_nodo -c JMSProviders
        	-o IBM_MessageSight -n jarsURL -v
ubicación_jar_cliente

    donde nombre_nodo es el nombre del nodo de integración (intermediario) (por ejemplo, IB9NODE) y ubicación_jar_cliente es la vía de acceso a los archivos JAR de cliente de IBM MessageSight (por ejemplo, C:\ImaClient\jms\lib).

  • Reinicie el nodo de integración.
