Parâmetros para o Padrão Filtro de Eventos

Configuração de Terminal do IBM MessageSight
Parâmetro de padrão Default Descrição
Servidor

O nome do host ou endereço IP do servidor IBM MessageSight
Porta

O número da porta do Terminal do IBM MessageSight
Filtragem de Eventos
Parâmetro de padrão Default Descrição
Tópico /PACEMAKER/#

O tópico usado pelo fluxo para assinar mensagens de entrada
Filtro contains($Body/Data/event/type,"alert")

Um valor XPath usado para filtrar mensagens de entrada
Saída de Evento Filtrada
Parâmetro de padrão Default Descrição
Nome do gerenciador de filas

O Gerenciador de Filas usado para armazenar mensagens filtradas
Nome da fila EVENT.IN

A Fila usada para armazenar mensagens filtradas
Identificador de erros
Parâmetro de padrão Default Descrição
Mensagem de erro necessária Selecionado

Especifica se uma mensagem de erro é necessária se ocorrer um erro no fluxo
Gerenciador de filas de erros

O Gerenciador de Filas usado para armazenar mensagens de erro
Fila de Erros ERROR

A Fila usada para armazenar mensagens de erro

Todos os parâmetros de padrão são listados com seus valores-padrão. Onde não houver padrão indicado, o valor padrão será uma sequência vazia.
