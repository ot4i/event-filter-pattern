Tarefas a Serem Concluídas antes da Aplicação do Padrão Filtro de Evento de Entrada do IBM MessageSight

Antes de aplicar o padrão para produzir uma instância padrão, conclua as seguintes etapas:

  • Copie os arquivos JAR do cliente IBM MessageSight na máquina do IBM Integration Bus.
  • Atualize o serviço configurável do IBM MessageSight com o local dos arquivos JAR do cliente IBM MessageSight executando o seguinte comando em uma janela Console de Integração IBM: 

    mqsichangeproperties node_name -c JMSProviders
        	-o IBM_MessageSight -n jarsURL -v client_jar_location

    em que node_name é o nome do nó de integração (broker) (por exemplo, IB9NODE) e client_jar_location é o caminho para os arquivos JAR do cliente IBM MessageSight (por exemplo, C:\ImaClient\jms\lib).

  • Reinicie o nó de integração.
