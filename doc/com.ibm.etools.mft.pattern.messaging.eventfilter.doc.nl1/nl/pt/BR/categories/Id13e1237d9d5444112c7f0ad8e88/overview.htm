IBM MessageSight

IBM MessageSight é um servidor de sistema de mensagens baseado em dispositivo projetado para manipular grandes números de clientes e dispositivos conectados, além de processar grandes volumes de mensagens com latência consistente.

Os clientes da Internet podem publicar mensagens no IBM MessageSight, que são encaminhadas para os serviços dentro da intranet da empresa. Com o uso do IBM Integration Bus, é possível preparar as mensagens para processamento por sistemas backend ou publicação adicional em outros clientes.

Também é possível usar o IBM Integration Bus para processar mensagens de sistemas backend para criar tópicos publicados no IBM MessageSight. Os clientes da Internet podem assinar tópicos específicos no IBM MessageSight para serem notificados de novas mensagens quando elas forem publicadas.

O diagrama mostra uma visão geral dos componentes e das conexões dentro de uma solução IBM MessageSight.

IBM MessageSight suporta sistemas de mensagens para publicar/assinar e de ponto a ponto. O IBM MessageSight é um broker para publicar/assinar baseado em assunto, de modo que o conteúdo das mensagens não é enfileirado ou alterado pelo IBM MessageSight e nenhum aplicativo baseado em usuário pode ser executado no IBM MessageSight.
