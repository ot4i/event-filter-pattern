Tâches à effectuer avant l'application du modèle Filtre d'événement entrant IBM MessageSight

Avant d'appliquer le modèle permettant de produire une instance du modèle, procédez comme suit :

  • Copiez les fichiers JAR du client IBM MessageSight sur votre machine IBM Integration Bus.
  • Mettez à jour le service configurable IBM MessageSight avec l'emplacement des fichiers JAR du client IBM MessageSight en exécutant la commande suivante dans une fenêtre de la console IBM Integration : 

    mqsichangeproperties node_name -c JMSProviders 
        	-o IBM_MessageSight -n jarsURL -v client_jar_location

    où node_name correspond au nom de votre noeud d'intégration (courtier) (par exemple, IB9NODE) et client_jar_location au chemin des fichiers JAR de votre client IBM MessageSight (par exemple, C:\ImaClient\jms\lib).

  • Redémarrez le noeud d'intégration.
