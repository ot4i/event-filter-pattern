应用 IBM MessageSight Inbound Event Filter 模式之前要完成的任务

在应用该模式以生成模式实例之前，请完成下列任务：

  • 将 IBM MessageSight 客户机 JAR 文件复制到 IBM Integration Bus 机器。
  • 通过在 IBM Integration Console 窗口中运行下列命令，用 IBM MessageSight 客户机 JAR 文件的位置更新 IBM MessageSight 可配置服务：

    mqsichangeproperties node_name -c JMSProviders 
        	-o IBM_MessageSight -n jarsURL -v client_jar_location

    其中 node_name 是集成节点（代理）名称（例如 IB9NODE），client_jar_location 是到 IBM MessageSight 客户机 JAR 文件的路径（例如 C:\ImaClient\jms\lib）。

  • 重新启动集成节点。
