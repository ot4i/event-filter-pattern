套用 IBM MessageSight Inbound Event Filter 型樣之前要完成的作業

在套用型樣以產生型樣實例之前，請完成下列步驟：

  • 將 IBM MessageSight 用戶端 JAR 檔複製到 IBM Integration Bus 機器。
  • 將 IBM MessageSight 可配置服務更新為 IBM MessageSight 用戶端 JAR 檔的位置，在 IBM Integration Console 視窗中執行下列指令：

    mqsichangeproperties node_name -c JMSProviders 
        	-o IBM_MessageSight -n jarsURL -v client_jar_location

    其中，node_name 是您的整合節點（分配管理系統）名稱（例如，IB9NODE），client_jar_location 是您的 IBM MessageSight 用戶端 JAR 檔的路徑（例如，C:\ImaClient\jms\lib）。

  • 重新啟動整合節點。
