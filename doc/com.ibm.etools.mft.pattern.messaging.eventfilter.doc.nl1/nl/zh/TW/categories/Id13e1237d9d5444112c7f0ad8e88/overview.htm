IBM MessageSight

IBM MessageSight 是一個軟體驅動裝置型傳訊伺服器，它設計用來處理大量連接用戶端與裝置，並處理具有一致延遲的大量訊息。

網際網路用戶可以將訊息發佈給 IBM MessageSight，接著再將其轉遞給公司內部網路中的服務。您可以使用 IBM Integration Bus 來準備後端系統處理的訊息，或進一步發佈給其他用戶端。

您還可以使用 IBM Integration Bus 來從後端系統處理訊息，以建立在 IBM MessageSight 上發佈的主題。網際網路用戶可以訂閱 IBM MessageSight 上的特定主題，以在發佈新訊息時得到通知。

IBM MessageSight 支援發佈/訂閱及點對點傳訊。IBM MessageSight 是主題型發佈/訂閱分配管理系統，因此 IBM MessageSight 不會查詢或變更訊息的內容，且不能在 IBM MessageSight 上執行使用者型應用程式。
