IBM MessageSight

IBM MessageSight ist ein gerätebasierter Messaging-Server, der für die Handhabung einer großen Anzahl an verbundenen Clients und Geräten entwickelt wurde und ein hohes Aufkommen an Nachrichten mit einer konsistenten Latenz verarbeiten kann.

Internet-Clients können Nachrichten in IBM MessageSight veröffentlichen, die anschließend an Services innerhalb des Intranets des Unternehmens weitergeleitet werden. Mit IBM Integration Bus können Sie die Nachrichten zur Verarbeitung durch Back-End-Systeme oder zur weiteren Veröffentlichung für andere Clients erstellen.

Mithilfe von IBM Integration Bus können Sie außerdem Nachrichten aus Back-End-Systemen verarbeiten, um Themen zu erstellen, die in IBM MessageSight veröffentlicht werden. Internet-Clients können Themen in IBM MessageSight subskribieren, damit sie bei der Veröffentlichung neuer Nachrichten benachrichtigt werden.

Das Diagramm zeigt eine Übersicht über die Komponenten und Verbindungen innerhalb einer IBM MessageSight-Lösung.

IBM MessageSight unterstützt sowohl Publish/Subscribe als auch das Punkt-zu-Punkt-Messaging. Da IBM MessageSight ein betreffsbasierter Publish/Subscribe-Broker ist, wird der Inhalt von Nachrichten nicht durch IBM MessageSight abgefragt oder geändert. Daher können auch keine benutzerbasierten Anwendungen in IBM MessageSight ausgeführt werden.
