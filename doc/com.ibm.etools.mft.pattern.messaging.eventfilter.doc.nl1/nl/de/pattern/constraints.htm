Einschränkungen bei der Verwendung des Musters IBM MessageSight Inbound Event Filter

Bei Verwendung dieses Musters gelten eine Reihe von Einschränkungen.

Das Muster IBM MessageSight Inbound Event Filter kann nur verwendet werden, wenn alle der folgenden Bedingungen erfüllt sind:

  • Ein IBM MessageSight-Gerät oder ein virtuelles Image ist verfügbar.
