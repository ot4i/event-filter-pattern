Parameter für das Muster Event Filter

IBM MessageSight-Endpunktkonfiguration
Musterparameter Standard Beschreibung
Server

Der Hostname oder die IP-Adresse des IBM MessageSight-Servers
Port

Die Portnummer des IBM MessageSight-Endpunkts
Ereignisfilterung
Musterparameter Standard Beispiel
Thema /PACEMAKER/#

Das Thema, das vom Fluss für die Subskription von eingehenden Nachrichten verwendet wird.
Filter contains($Body/Data/event/type,"alert")

Ein XPath-Wert, mit dem die eingehenden Nachrichten gefiltert werden.
Ausgabe des gefilterten Ereignisses
Musterparameter Standard Beschreibung
Name des WS-Managers

Der WS-Manager, der zum Speichern von gefilterten Nachrichten verwendet wird.
Warteschlangenname EVENT.IN

Die Warteschlange, die zum Speichern von gefilterten Nachrichten verwendet wird.
Fehlerbehandlung
Musterparameter Standard Beispiel
Fehlernachricht erforderlich Ausgewählt

Gibt an, ob eine Fehlernachricht erforderlich ist, wenn im Fluss ein Fehler auftritt.
Fehler-WS-Manager

Der WS-Manager, der zum Speichern von Fehlernachrichten verwendet wird.
Fehlerwarteschlange ERROR

Die Warteschlange, die zum Speichern von Fehlernachrichten verwendet wird.

Zu allen Musterparametern sind die Standardwerte angegeben. Wenn kein Standardwert angegeben ist, ist der Standardwert eine leere Zeichenfolge.
