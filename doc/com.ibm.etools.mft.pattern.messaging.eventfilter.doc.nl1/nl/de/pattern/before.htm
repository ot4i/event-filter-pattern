Tasks, die vor der Anwendung des Musters IBM MessageSight Inbound Event Filter ausgeführt werden müssen

Bevor Sie das Muster anwenden, um eine Musterinstanz zu erstellen, müssen Sie die folgenden Schritte ausführen:

  • Kopieren Sie die IBM MessageSight-Client-JAR-Dateien auf Ihre IBM Integration Bus-Maschine.
  • Aktualisieren Sie den konfigurierbaren IBM MessageSight-Service mit der Speicherposition der IBM MessageSight-Client-JAR-Dateien, indem Sie folgenden Befehl in einem IBM Integration Console-Fenster ausführen: 

    mqsichangeproperties Knotenname -c JMSProviders
        	-o IBM_MessageSight -n jarsURL -v Client-JAR-Position

    Dabei steht Knotenname für den Namen Ihres Integrationsknotens (Brokers) (zum Beispiel IB9NODE). Client-JAR-Position ist der Pfad Ihrer IBM MessageSight-Client-JAR-Dateien (zum Beispiel C:\ImaClient\jms\lib).

  • Starten Sie den Integrationsknoten erneut.
