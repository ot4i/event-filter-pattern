Tasks to complete before applying the IBM MessageSight Inbound Event Filter pattern

Before applying the pattern to produce a pattern instance, complete the following steps:

  • Copy the IBM MessageSight client JAR files to your IBM Integration Bus machine.
  • Update the IBM MessageSight configurable service with the location of the IBM MessageSight client JAR files by running the following command in an IBM Integration Console window: 

    mqsichangeproperties node_name -c JMSProviders 
        	-o IBM_MessageSight -n jarsURL -v client_jar_location

    where node_name is your integration node (broker) name (for example, IB9NODE) and client_jar_location is the path to your IBM MessageSight client JAR files (for example, C:\ImaClient\jms\lib).

  • Restart the integration node.
