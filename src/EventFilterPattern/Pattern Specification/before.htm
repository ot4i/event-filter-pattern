Tasks to complete before applying the IBM MessageSight Inbound Event Filter pattern

Before applying the pattern to produce a pattern instance, complete one or more of the following tasks, if required:

  • Copy the IBM MessageSight client jars files to your IBM Integration Bus machine.
  • Update the IBM MessageSight configurable service to set the location of the client jar files. This is achieved by running the following command:
    • mqsichangeproperties <node_name> -c JMSProviders -o IBM_MessageSight -n jarsURL -v <client_jar_location>
